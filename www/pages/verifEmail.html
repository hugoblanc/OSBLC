Attention

L'adresse email cpe est assez restrictif sur les emails reçus. Il est possible que l'email prenne entre 15 et 20 min pour arriver dans votre boîte mail... De plus l'email sera mis dans vos "pourriels". Pour voir l'email vous devez aller dans votre boîte mail puis dans l'onglet préférences. Ensuite dans la catégorie Message cliquez sur l'onglet "Configurer les options de pourriels". Enfin dans la deuxième question "Que faire des pourriels après qu'ils aient été déclarés pourriel ou légitime ?" Sélectionnez "Déplacer dans la boîte de réception".

Si vous avez redirigé vos mails vers une autre adresse, il se trouvera peut-être dans vos SPAM avec le préfixe 'XXX - SPAM'

Si vous ne recevez pas le mail contactez nous!

A noter que l'application n'est qu'en version alpha pour l'instant, il est possible et même probable qu'il y ait quelques problèmes :) On fera de notre mieux pour les corriger!
